## 5. 함꼐 해결한 문제  
>+ 로그아웃 버튼 클릭 시 mytoken을 지우지 못하는 현상($.cookie is not a function)이 나타나서 플러그인(<script type=text/javascript
            src=https://cdnjs.cloudflare.com/ajax/libs/jquery-cookie/1.4.1/jquery.cookie.min.js></script>) 추가를 했습니다.

>+ 라이브러리 동시 사용 시 html상 이미지, css 깨짐 현상 발생  
원인: Bulma, Bootstrap 라이브러리를 동시에 사용하다 보니  두 기능간 충돌  
해결: 기존 mainpage, mypage에 Bulma 모달 창 기능 삭제 후  html, css로  구현

 봉사활동 카드를 어떤 방식을 배치할 수 있을까?
 -> CSS에서 flex속성을 이용해서 카드를 배치했다.
     justify-content: space-between; 속성을 사용했을 때 
     마지막 줄의 봉사활동 카드가 2개일 경우 카드가 양 옆으로 정렬되는 문제는
     ::after를 사용해서 문제를 해결

 마감일을 현재 날짜에 맞게 보여줄 수 있을까?
 -> 서버에서 데이터를 받아올 때 DB에 있던 모집기간의 마지막 날과 현재 날짜를 빼서
     현재 날짜에 맞는 마감일 데이터를 받아올 수 있었다.
      python의 datetime 클래스를 사용했다. 

 봉사활동 카드를 마감일이 임박한 순서대로 보여줄 수 있을까?
 -> 서버에서 데이터를 받아올 때 정렬 함수를 사용해서 마감일 기준으로 데이터를 받아왔다. 
     rows = sorted(rows, key=(lambda x: x['deadline'])) 

1365 페이지에서 어떻게 크롤링할 수 있을까?
- bs4를 라이브러리를 이용

1365 사이트에서 인천광역시 연수구 항목을 선택한 결과를 크롤링해와야하는데 어떻게 하지?
- URL의 쿼리스트링에 값을 입력해서 request하면 해결된다는 점을 알아냄
- URL의 쿼리스트링에 어떤 값을 입력해야하는 지 찾아봄
- 인천광역시에 대한 코드번호, 연수구에 대한 코드번호가 있는 것을 확인하고 쿼리스트링을 이에 대해 맞춰 작성하여 URL을 완성

1365 사이트에서 조회한 결과 페이지가 여러 개인데, 여러 개 페이지를 어떻게 크롤링해오지?
- 쿼리스트링의 cPage라는 값에 페이지번호를 입력하면 해당 페이지로 이동한다는 점을 알아냄
- 페이지가 여러 개이면 반복문을 이용해 cPage의 값을 바꿔줌

크롤링해온 결과에서 필요한 데이터를 어떻게 뽑아오지?
- 1365 페이지에서 개발자도구를 연 다음, copy selector 기능을 이용해서 데이터가 있는 선택자를 가져옴
- 필요한 데이터만 얻을 수 있게 부가적으로 포함되어있는 html 태그 등을 제거함
- 추가로 필요할 경우, split 함수를 이용해서 데이터의 일부를 가져옴

페이지가 몇 개 있는 지 어떻게 알지?
- 1365 페이지에서 조회 결과 개수가 써있는 선택자에서 크롤링해옴
- 1365 사이트를 보면 한 페이지 당 10개의 목록을 보여주기 때문에 “개수 / 10” 을 계산하여 알아냄

마이페이지에서 메인페이지로 이동할 경우, 크롤링한 데이터가 중복되서 화면에 나타난다. 원인이 무엇일까?
- 메인페이지에서 마이페이지로 이동할 때 봉사 데이터가 담겨있는 리스트가 초기화되지 않았기 때문에 초기화되지 않은 상태에서 추가로 데이터를 넣었기 때문으로 예상됨
- 리스트에서 중복된 데이터를 제거하여 해결함
